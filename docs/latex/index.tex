\char`\"{}\+FMI Tables\char`\"{} is a simple implementation of parsing and editing a CSV-\/style table file, that stores strings, formulas and numbers.\hypertarget{index_autotoc_md1}{}\doxysection{Structure overview}\label{index_autotoc_md1}
The project is roughly divided into 3 main components\+: User Interface, Models and Generic \mbox{\hyperlink{index_fig1}{fig1}}.
\begin{DoxyItemize}
\item The User Interface is the messenger between a user and the underlying application. All input and output is handled here.
\item Models are the general classes that are used in the user interface and that do the value parsing.
\item Generic is a place for all code that is can be used independently from the project. Stuff like \mbox{\hyperlink{structPair}{Pair}}, \mbox{\hyperlink{classString}{String}} or \mbox{\hyperlink{classList}{List}}.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{Building the project}\label{index_autotoc_md2}
